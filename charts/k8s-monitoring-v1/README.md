# Kubernetes Monitoring Helm chart (v1) — Deprecated

The v1 (`1.x`) version of the `k8s-monitoring` Helm chart has been **deprecated**
and is no longer maintained. Its source has been removed from the `main` branch.

We strongly recommend migrating to the current major version of the
[`k8s-monitoring`](../k8s-monitoring) chart. See the
[migration guide](https://github.com/grafana/k8s-monitoring-helm/blob/main/charts/k8s-monitoring/docs/Migration.md)
for help moving from v1.

## Looking for the v1 chart or its documentation?

The full source and documentation for the last v1 release remain available on the
[`1.6-stable`](https://github.com/grafana/k8s-monitoring-helm/tree/1.6-stable/charts/k8s-monitoring-v1)
branch.
