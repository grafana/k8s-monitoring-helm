# Integration: etcd

This example demonstrates how to gather metrics from [etcd](https://etcd.io/).

## Values

```yaml
---
cluster:
  name: etcd-integration-cluster

destinations:
  - name: prometheus
    type: prometheus
    url: http://prometheus.prometheus.svc:9090/api/v1/write

integrations:
  etcd:
    instances:
      - name: etcd
        labelSelectors:
          app.kubernetes.io/component: etcd

alloy-metrics:
  enabled: true
```
